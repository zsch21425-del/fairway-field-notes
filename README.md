# Fairway Subaru — Field Notes for Wes

A small two-book library. This folder contains everything you need to put these guides online so anyone with the link can read them.

## What's in this folder

- **`index.html`** — the landing page (opens both guides)
- **`upstate-subaru-operator.html`** — the full career playbook
- **`working-the-gold-mine.html`** — the database mining guide
- **`START-HERE.txt`** — step-by-step instructions for publishing
- **`README.md`** — this file

## Quick preview

Double-click `index.html` right now to see how the finished site will look before you publish anything.

## How to publish (5 minutes)

Follow **`START-HERE.txt`** — it walks you through every click.

## After publishing

Your public URL will look like:

    https://YOUR-USERNAME.github.io/REPO-NAME/

Send that one link to anyone. It opens to the landing page with both guides.
